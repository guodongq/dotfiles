vimrc
====

## Prequirement

- git
- curl
- nodejs
- ctags
- ack

## Getting Started

```
git clone https://github.com/guodongq/vimrc ~/.vimrc
```

## Vim Integrated
```
ln -sv ~/.vimrc ~/.vim
```

## NeoVim Integrated

```
ln -sv ~/.vimrc ~/.config/nvim
```

## Tmux Integrated

```
ln -sv ~/.vimrc/.tmux.conf ~/.tmux.conf
```

## Shortcuts

Some of the shortcuts I use frequently. In the following shortcuts, <leader> represents `SPACE`.

| Shortcuts   | Mode   | platform        | Description                |
| ---         | ---    | ---             | ---                        |
| <leader> ft | Normal | Linux/MacOS/Win | 打开文件树                 |
| <leader> fo | Normal | Linux/MacOS/Win | 当前打开的文件对应的文件树 |
| <leader>sa | Normal | Linux/MacOS/Win | 搜索匹配的内容 |
| ctrl+p | Normal | Linux/MacOS/Win | 搜索文件 |
| <leader>h | Normal | Linux/MacOS/Win | 打开开始页面 |
| <leader>tt | Normal | Linux/MacOS/Win | 打开标签页 |
| <leader>wc | Normal | Linux/MacOS/Win | 选择窗口 |
| ss | Normal | Linux/MacOS/Win | 快速跳转 |
| <leader>wm | Normal | Linux/MacOS/Win | 窗口最大化 |
